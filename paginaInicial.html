Eduardo Santos Pontes ,

Me chamo Eduardo Pontes e sou um desenvolvedor em formação, atualmente cursando Desenvolvimento de Sistemas pelo SENAI. Tenho me dedicado intensamente ao aprendizado de tecnologias voltadas ao desenvolvimento web, com foco em lógica de programação, banco de dados, versionamento de código e práticas de desenvolvimento ágil.

Paralelamente, estou me especializando na trilha Front-End pela plataforma Upper, onde venho aprimorando minhas habilidades em HTML, CSS, JavaScript e construção de interfaces modernas, acessíveis e responsivas.

Além disso, estou me preparando para ingressar na graduação em Engenharia de Software, com o objetivo de aprofundar meu conhecimento em arquitetura de sistemas, engenharia de requisitos e desenvolvimento full stack.

Tenho como características profissionais a proatividade, facilidade de aprendizado, organização e comprometimento com a qualidade do que desenvolvo. Busco constantemente novos desafios, visando evoluir tanto tecnicamente quanto como membro de equipe.

Meu objetivo é atuar como desenvolvedor em projetos que façam a diferença, utilizando a tecnologia para resolver problemas reais e impactar positivamente o mundo ao meu redor.

Minhas Habilidades

HTML5

Tenho domínio em estruturação de páginas com HTML5, criando marcações semânticas e acessíveis. Utilizo boas práticas para garantir performance, SEO básico e compatibilidade com diferentes navegadores e dispositivos.

CSS3

Sou capaz de desenvolver interfaces modernas e responsivas com CSS3. Trabalho com flexbox, grid layout, media queries e animações para entregar experiências visuais atrativas e fluidas.

JavaScript

Possuo conhecimento em JavaScript para desenvolver interações dinâmicas no front-end. Utilizo lógica de programação, manipulação do DOM, eventos e funções assíncronas para consumir APIs e criar comportamentos interativos.

SQL Server

Tenho noções sólidas em banco de dados relacionais, com foco no SQL Server. Consigo criar, consultar, atualizar e gerenciar tabelas utilizando comandos como SELECT, JOIN, INSERT, UPDATE e DELETE.

Git & GitHub

Versiono meus projetos com Git e GitHub, mantendo organização no código por meio de commits bem descritos, uso de branchs e integração com equipes via repositórios remotos.

C#

Estou aprendendo a desenvolver com C#, especialmente para aplicações backend e integração com banco de dados. Tenho experiência prática com classes, métodos, estruturas de repetição, coleções e orientação a objetos.
